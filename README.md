# todo-app
JavaScript To-Do App with edit, filters, dark mode and localStorage
- Add, Edit, Delete tasks
- Mark tasks as complete
- Filter (All / Completed / Pending)
- Dark mode
- Categories
- Local storage (data persists after refresh)
